表操作

createTable

描述

创建一个新的数据表。

参数

参数 类型 默认值 描述
tableName string - 表名
options object {} 创建表的选项

返回值

Promise<void>

示例

import { createTable } from 'expo-lite-db-store';

// 创建一个简单的表
await createTable('users');

// 使用选项创建表
await createTable('users', {
  encrypted: true, // 启用加密
  autoIndex: true, // 自动创建索引
  indexes: ['id', 'email'] // 指定索引字段
});

deleteTable

描述

删除一个数据表。

参数

参数 类型 默认值 描述
tableName string - 表名

返回值

Promise<void>

示例

import { deleteTable } from 'expo-lite-db-store';

await deleteTable('users');

hasTable

描述

检查数据表是否存在。

参数

参数 类型 默认值 描述
tableName string - 表名

返回值

Promise<boolean>

示例

import { hasTable } from 'expo-lite-db-store';

const exists = await hasTable('users');
console.log(exists); // true 或 false
